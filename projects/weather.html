Overview

A weather application displaying real-time and 5-day forecasts, using Axios for API requests and conditional rendering for weather icons.

Key Features

  • Third-party weather API integration
  • Error handling for invalid searches
  • Conditional UI updates
  • Responsive design
  • Deployed on Vercel

Core Code Example

const fetchWeather = async (city) => {
  try {
    const response = await axios.get(
      \`https://api.weatherapi.com/v1/forecast.json?key=API_KEY&q=\${city}&days=5\`
    );
    setWeather(response.data);
  } catch (error) {
    setError("City not found");
  }
};
