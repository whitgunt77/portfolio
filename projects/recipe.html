Overview

A full-stack recipe application built with React and Node. It allows users to create, edit, and delete recipes while browsing community submissions.

Key Features

  • Responsive UI with an intuitive card-based layout
  • Form validation using Yup
  • CRUD operations using Axios
  • Jest test suites for component validation
  • Deployed on Vercel

Core Code Example

const handleSubmit = async (values) => {
  try {
    await axios.post("/api/recipes", values);
    alert("Recipe added!");
  } catch (error) {
    console.error(error);
  }
};
